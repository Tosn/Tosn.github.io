# 5.语句

  • 条件（conditional）语句，Javascript解释器可以根据一个表达式的值来判断是执行还是跳过这些语句，如if语句和switch语句
  • 循环（loop）语句，可以重复执行语句，如while和for语句
  • 跳转（jump）语句，可以让解释器跳转至程序的其他部分继续执行，如break、return和throw语句
