# 踩坑记录

注意事项：

1、vue文件引用组件，使用动态按需引入会无效

// wrong
components: {
  HelloWorld: () => import('@/components/HelloWorld')
}
// right
import HelloWorld from '@/components/HelloWorld'
components: {
  HelloWorld
}

2、使用sass需要安装在开发插件上

// wrong
npm install sass -S
// right
npm install sass -D
